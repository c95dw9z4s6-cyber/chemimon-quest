START GUIDE

最初に確認：パソコン・スマホでの始め方

ゲーム中の操作はボタンを押すだけです。クイズ中は戦闘・エナジー回復・タイマーがすべて停止します。

🖥️

パソコン

  1. ZIPを右クリックして展開します。
  2. index.htmlをChrome・Edge・Safariなどで開きます。
  3. ユニットを押してクイズに正解すると召喚できます。
📱

スマートフォン

  1. SafariまたはChromeでゲームのURLを開きます。
  2. 見やすさのため横向きを推奨します。
  3. ブラウザーの「ホーム画面に追加」を使うとアプリ風に起動できます。

Androidでは展開したindex.htmlを直接開ける場合もあります。iPhone/iPadではWeb上のURLから開く方法が最も確実です。

化学相性（v1.1）

大学受験で扱う代表的な酸・塩基だけを属性として使用します。表示上の「強酸・弱酸・強塩基・弱塩基・相性なし」を見て戦い方を考えてください。

強酸 → 弱酸：与ダメージ1.5倍 弱酸 → 強酸：約0.67倍 強塩基 → 弱塩基：与ダメージ1.5倍 弱塩基 → 強塩基：約0.67倍 酸 ↔ 塩基：双方0.75倍 同じ強さ同士・中性・相性なし：等倍

例：HCl（塩酸）・HNO₃（硝酸）・H₂SO₄（硫酸）は強酸、CH₃COOHとNH₄⁺は弱酸、NaOHは強塩基、NH₃とSO₄²⁻は弱塩基として扱います。受験で頻出する代表物質に絞った学習用分類です。NO₃⁻・CuSO₄・Na₃AlF₆は相性なしです。
