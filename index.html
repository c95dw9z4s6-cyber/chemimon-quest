HOW TO PLAY

遊び方

1. 召喚ユニットを押し、クイズ正解で召喚。
2. 防衛30秒ごとに敵が追加。5分で最終ウェーブ。
3. 成長敵撃破でコイン・EXP。難問で解放と研究。
4. クリア敵拠点を破壊。Stage 1クリアでStage 2解放。
属性と相性
強酸→弱酸：弱酸の遊離・1.5倍強塩基→弱塩基：弱塩基の遊離・1.5倍酸↔塩基：中和・0.75倍逆相性：約0.67倍
ユニットとステージ進行

ステージごとに編成・解放順・能力が変わります。切り替えても、前のステージのコイン・解放・強化は保存されます。

セーブとオンライン

設定からセーブ・ロードできます。タブを閉じても通常は残りますが、履歴・サイトデータを消すと失われることがあります。ユーザー名は作成者へLINEで送ってください。

画面の記号

⚔攻撃　◎射程　♥HP　➤速度　⏱攻撃間隔　●コイン　✦EXP

QUESTION MODE

出題範囲を選択

召喚・解放・研究強化で出る問題の範囲です。変更すると現在のステージは第1ウェーブから再開しますが、そのステージのコイン・解放・強化は残ります。

STAGE SELECT

ステージを選択

ステージごとにコイン・ユニット解放・研究強化は独立します。化学レベル、実績、累計記録、復習問題は共通です。切り替えると戦闘は第1ウェーブから始まります。
