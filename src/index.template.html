HOW TO PLAY

遊び方

1. 召喚ユニットを押し、クイズ正解で召喚。
2. 防衛30秒ごとに敵が追加。5分で最終ウェーブ。
3. 成長敵撃破でコイン・EXP。難問で解放と研究。
4. クリア敵拠点を破壊。Stage 1→2→3→4→5→6→7の順に解放。Stage 5は「5の倍数・難関」です。
属性と相性
味方の強酸→敵の「弱酸由来」：1.4倍味方の強塩基→敵の「弱塩基由来」：1.4倍弱酸・弱塩基そのものには遊離補正なし敵側は相性補正なし

弱酸の遊離は、強酸が弱酸の塩に由来する陰イオンへH⁺を与えて弱酸を生じさせる反応です。例：CH₃COO⁻ + H⁺ → CH₃COOH。弱塩基の遊離は、強塩基が弱塩基の塩に由来する陽イオンへ作用して弱塩基を生じさせる反応です。例：NH₄⁺ + OH⁻ → NH₃ + H₂O。

酸と塩基がぶつかると「中和」または「酸塩基反応」の演出が出ますが、現在は演出のみで、中和によるダメージ倍率の変化はありません。酸・塩基の強弱は電離の程度を表し、危険性や腐食性を直接表すものではありません。

ユニットとステージ進行

ステージごとに編成・解放順・能力が変わります。同じ化学式が別ステージに登場しても、ステージ固有の別ユニットとして扱われ、レベルや解放状況は持ち越されません。前のステージの進行は保存されます。

高性能ヒーラー

後半で解放されるヒーラーは、傷ついた味方1体を大きく回復します。自分自身は回復せず、盾兵など最大HPの高い味方を優先しやすい単体支援型です。強化レベルに応じて回復量も伸びます。

Stage 3以降の研究カード

Wave 3・6・9の終了後に3枚から1枚を選び、そのバトル中だけ味方を強化します。Stage 1・2には登場しません。

飛行型

✈ FLYING 通常の地上近接攻撃を受けません。弓兵・速射・範囲・飛行に加え、BOSSからも攻撃を受けます。対空攻撃には弱い一方、v4.5以降は時間経過による自傷ダメージはありません。

5の倍数・難関ステージ

Stage 5、10、15…は前後より難しい節目のステージとして表示されます。今後はStage 5が6・7より難しく、Stage 8はStage 5より難しい程度の曲線を基準に調整します。

Stage 6・弱酸の遊離

Stage 6の敵はすべて弱酸の塩に由来する陰イオンです。第5ユニットの強酸が攻撃すると、弱酸の遊離として1.4倍になります。特別な解放イベントはなく、Stage 5クリア後に通常どおり解放されます。BOSSは予告後に弱酸由来イオンを召集します。

Stage 7・弱塩基の遊離

Stage 7の敵はすべて弱塩基の塩に由来する陽イオンです。第5ユニットの強塩基が攻撃すると、弱塩基の遊離として1.4倍になります。Stage 6クリア後に通常どおり解放され、BOSSは予告後に弱塩基由来イオンを召集します。

Stage 8・蓄積急襲BOSS

Stage 8のWave 10では、O₃ BOSSの出現演出と同時に場の味方ユニットがすべて倒されます。BOSSは第二形態を持たずHPは一般敵級ですが、移動速度60でLv.MAX強襲型の速度50を上回ります。Energy上限をLv.3以上へ拡張し、125以上を蓄えてからBOSSへ入ると、直後に盾役と攻撃役を再展開できます。全味方消去・速度・役割はゲーム上の表現であり、O₃の実際の反応速度や作用を表しません。

ポーズとウェーブ1からの再開

戦闘中のポーズ画面から、現在の挑戦を諦めて同じステージのウェーブ1へ戻れます。場の味方・敵、ウェーブ内の進行、その挑戦中の研究カードはリセットされますが、所持コイン、解放、恒久強化、実績、学習記録は維持されます。

専用の倍速試験

研究強化の「倍速試験」を選び、専用の難問に正解すると、戦闘が5分間1.5倍速になります。1.5倍速中にもう一度倍速試験へ正解すると2.25倍速になりますが、残り時間は延長されません。不正解の場合は30秒間、倍速試験へ再挑戦できません。ページを再読み込みしても制限は継続します。ユニット解放・強化など、ほかの難問では倍速になりません。クイズ、設定、ランキング、ポーズなどで戦闘が止まっている間は残り時間も減りません。

学習記録・練習

設定から学習記録、分野別練習、難問練習、間違い復習を利用できます。難問の段階ヒントはステージのコインを消費します。

共通テスト型・実戦問題

Stage 1クリア後、設定の「実戦問題」から共通資料を使う5問構成の大問へ挑戦できます。3問以上正解で合格し、次のバトル1回だけ有効なボーナスを1つ選べます。

問題の重複防止と出典

同じ問題は直近30問、文章が近い類題は直近20問、同じ計算パターンは直近8問を避けて出題します。候補が少ない場合だけ段階的に制限を緩めます。難問には「出典・参考資料」を表示し、公式公開問題は転載せず、数値・問題文・選択肢を独自に作成しています。

難問・大学二次型の出題範囲

「化学基礎中心」では化学基礎に加え、電気分解と結晶を中心に出題します。「理論化学全範囲（予習）」を選ぶと、気体、蒸気圧、反応熱、反応速度、化学平衡も出題します。有機化学は現在の問題集には含まれていません。

アプリ・オフライン・更新

設定の「アプリ・更新」からホーム画面への追加方法と最新版を確認できます。一度オンラインで読み込んだ後は、通信が切れてもゲーム本体を起動できます。ランキング・要望の閲覧と送信には通信が必要です。新しい版が準備できた場合は、セーブ後にゲーム操作を止めて必須更新します。更新取得に失敗した場合は再試行できます。

セーブとオンライン

設定からセーブ・ロードできます。さらに「セーブコード」で別端末へゲーム進行・学習記録・復習記録を移せます。コードは暗号化されていないため他人へ送らないでください。ユーザー名は作成者へLINEで送ってください。

QUESTION MODE

出題範囲を選択

召喚・解放・研究強化で出る問題の範囲です。既定では化学基礎＋電気分解＋結晶だけを出題し、ボイルの法則・エンタルピーなどは「理論化学全範囲（予習）」を選んだ場合だけ出題します。変更すると現在のステージは第1ウェーブから再開しますが、コイン・解放・強化は残ります。

STAGE SELECT

ステージを選択

ステージごとにコイン・ユニット解放・研究強化は独立します。化学レベル、実績、累計記録、復習問題は共通です。切り替えると戦闘は第1ウェーブから始まります。
