HOW TO PLAY

遊び方

1. 召喚ユニットを押し、クイズ正解で召喚。
2. 防衛30秒ごとに敵が追加。5分で最終ウェーブ。
3. 成長敵撃破でコイン・EXP。難問で解放と研究。
4. クリア敵拠点を破壊。Stage 1→2→3→4→5の順に解放。Stage 5は「5の倍数・難関」です。
属性と相性
味方の強酸→敵の弱酸：1.4倍味方の強塩基→敵の弱塩基：1.4倍その他：1.0倍敵側は相性補正なし
ユニットとステージ進行

ステージごとに編成・解放順・能力が変わります。同じ化学式が別ステージに登場しても、ステージ固有の別ユニットとして扱われ、レベルや解放状況は持ち越されません。前のステージの進行は保存されます。

高性能ヒーラー

後半で解放されるヒーラーは、傷ついた味方1体を大きく回復します。自分自身は回復せず、盾兵など最大HPの高い味方を優先しやすい単体支援型です。強化レベルに応じて回復量も伸びます。

Stage 3以降の研究カード

Wave 3・6・9の終了後に3枚から1枚を選び、そのバトル中だけ味方を強化します。Stage 1・2には登場しません。

飛行型

✈ FLYING 通常の地上近接攻撃を受けません。弓兵・速射・範囲・飛行に加え、BOSSからも攻撃を受けます。対空攻撃には弱く、飛行ユニットは時間とともにHPが減少します。

5の倍数・難関ステージ

Stage 5、10、15…は前後より難しい節目のステージとして表示されます。今後はStage 5が6・7より難しく、Stage 8はStage 5より難しい程度の曲線を基準に調整します。

専用の倍速試験

研究強化の「倍速試験」を選び、専用の難問に正解すると、戦闘が5分間1.5倍速になります。1.5倍速中にもう一度倍速試験へ正解すると2.25倍速になりますが、残り時間は延長されません。ユニット解放・強化など、ほかの難問では倍速になりません。クイズ、設定、ランキング、ポーズなどで戦闘が止まっている間は残り時間も減りません。

学習記録・練習

設定から学習記録、分野別練習、難問練習、間違い復習を利用できます。難問の段階ヒントはステージのコインを消費します。

問題の重複防止と出典

同じ問題は直近30問、文章が近い類題は直近20問、同じ計算パターンは直近8問を避けて出題します。候補が少ない場合だけ段階的に制限を緩めます。難問には「出典・参考資料」を表示し、公式公開問題は転載せず、数値・問題文・選択肢を独自に作成しています。

大学二次型の出題範囲

化学基礎、電気分解、化学範囲の結晶だけです。ヘンリーの法則、蒸気圧、化学平衡、反応速度、有機化学などは出題しません。

アプリ・オフライン・更新

設定の「アプリ・更新」からホーム画面への追加方法と最新版を確認できます。一度オンラインで読み込んだ後は、通信が切れてもゲーム本体を起動できます。ランキング・要望の閲覧と送信には通信が必要です。新しい版が準備できた場合は通知を表示し、「更新する」を押したときだけ切り替えます。

セーブとオンライン

設定からセーブ・ロードできます。さらに「セーブコード」で別端末へゲーム進行・学習記録・復習記録を移せます。コードは暗号化されていないため他人へ送らないでください。ユーザー名は作成者へLINEで送ってください。

QUESTION MODE

出題範囲を選択

召喚・解放・研究強化で出る問題の範囲です。既定では化学基礎＋電気分解＋結晶だけを出題し、ボイルの法則・エンタルピーなどは「理論化学全範囲（予習）」を選んだ場合だけ出題します。変更すると現在のステージは第1ウェーブから再開しますが、コイン・解放・強化は残ります。

STAGE SELECT

ステージを選択

ステージごとにコイン・ユニット解放・研究強化は独立します。化学レベル、実績、累計記録、復習問題は共通です。切り替えると戦闘は第1ウェーブから始まります。

BATTLE RESEARCH

研究カードを1枚選択

このバトル中だけ有効です。選択すると戦闘を再開します。

Stage 3以降、Wave 3・6・9の終了後に選択できます。選んだ効果は同じ種類でも重なります。Stage 5は飛行型を含む難関総合戦です。
