# Chemion Quest v5.7 設定整理レポート

## 目的

v5.6までの設定画面は、学習、練習、更新情報、音、低電力、セーブ、要望、初期化などが一つの長い一覧に並んでいた。v5.7では既存機能を削除せず、目的別に整理してスマートフォンで探しやすくする。

図鑑機能は必要性、既存機能との重複、UI負担を再検討し、実装前に廃止した。図鑑用のセーブ項目やコードは追加していない。

## 新しい分類

1. 学習・プレイ
   - 間違い復習
   - 学習記録
   - 練習モード
   - 実戦問題
   - 遊び方
   - 操作チュートリアル

2. 音・表示
   - 効果音
   - BGM ON/OFF・音量
   - 低電力モード

3. セーブ・アカウント
   - セーブ
   - ロード
   - セーブコード
   - ユーザー名変更
   - ユーザー名の安全案内

4. サポート・情報
   - お知らせ
   - アップデート履歴
   - 要望一覧
   - 要望送信
   - アプリ・更新

5. データ管理
   - 学習データ初期化
   - セーブ削除

## UI方針

- 分類はHTMLの`details`要素による折りたたみ式とした。
- よく使う音・表示だけを初期状態で開く。
- 初期化と削除は赤系の注意領域へ分離する。
- 危険操作の実際の確認処理は従来のまま維持する。
- 既存のボタンIDとイベント処理を維持し、機能回帰を避ける。
- 低電力モードでは新しい設定セクションの影も無効化する。

## 互換性

- セーブバージョン31を維持する。
- 新しいセーブ項目は追加しない。
- Stage 1〜9、840問、戦闘値、報酬、BGM、低電力設定は変更しない。
- 既存の設定ボタンIDと`data-settings-view`を維持する。

## 受け入れ条件

- 5分類がすべて存在する。
- 音・表示が初期状態で開いている。
- 学習データ初期化とセーブ削除がデータ管理内にある。
- 既存の設定機能がすべて操作できる。
- ブラウザ試験で低電力、BGM音量、学習データ初期化を確認する。
- 問題、化学、セーブ移行、公開ZIPの既存監査をすべて通す。
