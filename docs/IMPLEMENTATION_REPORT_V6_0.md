# Chemion Quest v6.0 実装レポート

作成日: 2026-07-18

## 実装基準

- 唯一の実装ベースは、正式なv5.95コミット `d5ebfc763010063961e2c1143bb9c4143eecf640` です。
- v5.95のStage 1〜9、ゲストアシスト、学習記録、実績、ランキング、PWA、BGM、問題数を維持しています。
- v6.1以降のPt希少敵、通常変異体、新しい調製物、装備、クリティカル、レア研究カード、Stage間素材、対人戦、マルチ要素は実装していません。

## Stage 10

- 「黄金王・Au反応区」を追加しました。全10 Waveで、Wave 10はAu単体です。
- 論理戦場幅を既存Stageの1.30倍にし、描画領域を拡大せず座標変換で実現しました。
- 味方は既存のHNO₃、HCl、Al、Fe、H₂Oの5種を再利用しました。
- 通常敵は既存のK⁺、Cu²⁺、Al₂O₃、NO₃⁻、Cl₂を再利用しました。
- Stage 9クリア済みセーブは、ロード時にStage 10を解放します。
- Au撃破をStage 10の勝利条件とし、敵拠点破壊は要求しません。

## 王水調製

- HNO₃ 1体とHCl 3体が半径90の範囲に1秒間安定して存在した場合、王水1体へ調製します。
- 調製はトランザクションとして管理し、素材の二重消費、同時多重生成、追加Energy消費を防止しています。
- 調製後HPは4素材の残HP率の平均を引き継ぎます。
- 王水はStage 10専用かつ独立したLv1〜10強化です。
- 解放価格は220 coin、Lv2〜10の費用は70、95、125、165、215、275、345、425、515 coinです。
- HP成長率13%、攻撃成長率18%、攻撃間隔4.8秒、1攻撃6ヒット、ヒット間隔0.14秒です。
- 数値は既存終盤coin経済から独立させ、低LvだけでAuを容易に倒せず、Lv3〜4と支援運用で突破可能になるよう調整しました。

## Au

- HP 10000、物理攻撃138、移動速度2.8、攻撃間隔5.4秒です。
- 通常の化学攻撃を80%軽減します。Feの物理攻撃と王水のAu専用攻撃は軽減を迂回します。
- 金箔攻撃は20秒間隔、警告1.7秒、基礎ダメージ88、押し戻し84です。
- 金塊攻撃は警告0.65秒、範囲48、周辺倍率0.34、押し戻し28です。
- 28秒間隔で既存敵から護衛2体を召喚します。
- 出現時は味方を保持し、味方の攻撃弾のみ消去して3秒の保護演出後に戦闘へ移行します。

## 初接触・BGM・セーブ

- 王水がAuへ初めて接触した際、ダメージ適用前に一度だけ専用演出を行います。
- 初接触済み状態を保存し、ロード後や再戦で重複させません。
- Au戦開始後に指定音源V16へ切り替えます。既存のBGM ON/OFF、音量、非表示時停止、オフライン再生を維持しています。
- セーブ形式を32へ更新しました。v5.95形式31および旧v5.0系セーブを直接読み込みます。
- 王水解放、王水Lv、初接触済み、Stage 10統計を追加し、既存のゲストアシスト使用済みフラグを保持します。
