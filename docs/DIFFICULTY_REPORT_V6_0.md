# Chemion Quest v6.0 難易度レポート

作成日: 2026-07-18

## 目標

- 王水なしでもFe中心の積極運用には低確率の突破余地を残す。
- 王水Lv1だけでAuを簡単に突破させない。
- 王水Lv3〜4と既存ユニット支援で、十分な操作を行えば過半数程度を突破可能にする。
- 王水Lv10を放置勝利にはせず、積極運用では明確に強くする。
- 長時間の膠着を発生させない。

各条件240試行です。

全条件で膠着率は0%でした。

## 数値採用理由

- Au HP 10000と化学80%軽減により、王水の価値を明確にしつつFeの物理攻撃を残しました。
- 王水の攻撃12、Lvごとの攻撃成長18%、4.8秒間隔・6分割は、低Lvの即時突破を防ぎ、高Lvの投資効果を出す組合せです。
- Auの28秒ごとの護衛召喚は、単体ボスへの火力集中だけでなく前線維持を要求します。
- 1.30倍の論理戦場と低速Auにより、準備時間を与えつつ、護衛と二種の物理攻撃で放置を防ぎます。
- 王水解放220 coinと段階費用は、v5.95の既存強化を変更せず、Stage 10専用の追加投資として設定しました。

## 制約と限界

シミュレーションは固定シードの1秒単位消耗モデルです。プレイヤー人口の実勝率予測ではありません。レーン単位のターゲット選択、個々のクイズ解答時間、端末性能差は再現しません。実ランタイムの6ヒット間隔、調製、Auフェーズ、セーブ復元、攻撃属性は別のブラウザ統合テストで検証しています。

## 判定

目標条件をすべて満たし、難易度テストは合格です。Stage 1〜9の数値は変更していません。
