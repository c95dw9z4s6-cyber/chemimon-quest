# Chemion Quest v5.1 学習データ初期化レポート

## 目的

偶然正解・適当な回答などが学習履歴へ混ざった場合に、ゲーム進行を失わず、学習記録と復習スケジュールだけを未学習状態から取り直せるようにしました。

## 入口

- 設定画面の「学習データ初期化」
- 「学習記録・習熟度」画面下部の「学習データを初期化」

## 初期化されるデータ

- 正解・不正解回数、正答率、現在・最高連続正解
- 分野別の正誤履歴と習熟度
- 問題ごとの回答回数、連続正解、誤答回数
- 最終回答日時、次回復習日時、復習間隔
- 間違い復習一覧
- 現在の起動中に保持している直近出題履歴

## 維持されるデータ

- コイン、Stage進行、Stage内セーブ
- 解放済みユニット、ユニット強化
- Energy上限、実績、ニックネーム、設定
- 実戦問題の合格・初回報酬記録

学習記録画面に表示される実戦問題の回答数・正答率は初期化されますが、初回報酬を再取得することはできません。

## 誤操作対策

専用確認画面で「初期化する」と完全一致する文字を入力するまで、実行ボタンは無効です。実行後は学習記録画面へ移動し、0問・未学習になったことをその場で確認できます。

## 検査

- 学習・復習用localStorageだけが削除されることをChromiumで確認
- コイン、Stage、解放、強化、実績、実戦問題進行が変化しないことを確認
- 初期化後に全体0/0問・全分野未学習として表示されることを確認
- v5.0以前の機能監査、問題検査、化学監査、Stage 6〜8のブラウザー試験を維持
