# Chemion Quest v4.45 修正レポート

## 要望管理権限

- `requestAdmins`コレクションと製作者向け管理者ID設定を廃止しました。
- 匿名認証を含むログイン済みの全ユーザーが、全要望を「実装済み／検討中」へ変更できます。
- ログイン済みの全ユーザーが全要望を削除できます。
- 削除前には、元に戻せないことを示す確認画面を表示します。
- Firestoreでは状態変更可能なフィールドを`status`、`implementedAt`、`implementedBy`だけに制限します。

## v4.4から維持する修正

- 飛行型を従来より42px上へ表示。
- Stage 5実績を`stage5Clears`で判定し、旧セーブを自動補完。
- 再生産クールタイムの表示と召喚可否を同期。
- セーブバージョン31、問題数、化学相性、ボス演出は変更なし。
