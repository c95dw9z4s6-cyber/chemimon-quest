# Chemion Quest v4.6 Stage 7レポート

## Stage 7「弱塩基遊離区」

- Stage 6クリア後に通常どおり解放。
- 全11Wave。
- 敵は弱塩基の塩に由来する陽イオン100％。
- 第5ユニットKOH（強塩基）が全敵へ1.4倍。
- 強塩基なしでも、十分な強化と編成でクリア可能。

## BOSS

- N₂H₅⁺／ヒドラジニウムイオン。
- 約20秒ごとに2秒の予告。
- 予告後、弱塩基由来イオンを2〜3体召集。

## 化学対応

- NH₄⁺ + OH⁻ → NH₃ + H₂O
- CH₃NH₃⁺ + OH⁻ → CH₃NH₂ + H₂O
- N₂H₅⁺ + OH⁻ → N₂H₄ + H₂O

表示する全遊離反応式は原子数・電荷を監査します。
