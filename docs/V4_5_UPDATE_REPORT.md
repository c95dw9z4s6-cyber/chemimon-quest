# Chemion Quest v4.5 追加・修正レポート

## 半反応式

- 基本問題40問、難問30問を追加。
- 酸性条件・塩基性条件、電子の位置、電子係数、電子mol、2つの半反応式の係数比を扱います。
- 基本520問、難問280問、実戦40小問です。

## 飛行型

- 味方・敵・BOSSの時間経過自傷を廃止。
- 近接無効、対空攻撃への弱さ、表示高度は維持。

## Stage 6

- Stage 5クリア後に通常どおり解放。
- 敵は弱酸由来陰イオン100％。
- 第5ユニットHCl（強酸）が全敵へ1.4倍。
- BOSS PO₄³⁻は約20秒ごとに予告し、弱酸由来イオンを2〜3体召集。
- 強酸なしでも、十分な強化と編成でクリア可能な設定。
