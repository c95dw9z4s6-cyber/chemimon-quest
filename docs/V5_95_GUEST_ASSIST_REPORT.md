# Chemion Quest v5.95 ゲストアシスト実装報告

## 実装範囲

- 完成版v5.9を唯一の実装ベースとし、Stage 1〜9・1000問・BGM・学習・設定・公開基盤を維持した。
- 設定画面のアクセスコードから、確認画面を経てゲストアシストを有効化できるようにした。
- コードは前後空白を除去し、大文字・小文字を区別せず `easy` と照合する。
- 有効中のゲーム内クイズは問題画面を表示せず、内部で成功結果へ進める。
- 召喚・解放・研究強化などの進行結果は成功させる一方、学習記録、正答率、連続正解、復習、実績、練習・実戦成績、完全正解報酬を更新しない。
- 有効状態と使用履歴をセーブへ保存する。使用履歴は解除後も維持し、そのセーブからのランキング送信を停止する。
- 既存セーブでは初期OFFとし、セーブバージョン31は変更しない。
- Firebaseルール・Firebaseサービスには変更を加えない。

## 公開表示

- 有効中のみ、小さな「ゲストアシスト中」表示を出す。
- トップ画面、遊び方、アップデート履歴には機能の詳細を掲載しない。
- v5.95の一般向け更新表示は公開基盤・内部機能の更新として案内する。

## 検証

- 静的構文、生成物同期、620基本問題・340難問・40実戦小問を検証した。
- v5.9までの全必須機能とStage 1〜9を回帰検証した。
- ブラウザで誤コード、確認、キャンセル、有効化、内部自動成功、統計除外、保存・ロード、解除、通常クイズ復帰、ランキング送信停止を検証した。
- v5.0相当のセーブバージョン31データからv5.95へ直接移行し、ゲストアシストが初期OFFになることを検証した。
- Service Workerがv5.95の新キャッシュを使用し、旧版キャッシュを削除することを検証した。
- PC 1280px、スマートフォン390pxで横方向の表示崩れがないことを検証した。
- 公開ZIPについて、完全性、欠落、余分なファイル、パストラバーサル、版不一致、破損を検証した。
