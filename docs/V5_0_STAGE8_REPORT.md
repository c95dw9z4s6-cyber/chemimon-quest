# Chemion Quest v5.0 Stage 8実装レポート

## Stage 8「蓄積急襲区」

- Stage 7クリア後に通常解放
- 全11Wave
- 高速・飛行・盾・回復・範囲を含む酸化還元混成
- Wave 10 BOSSはO₃／オゾン

## BOSS仕様

- 第二形態なし
- HP 420（一般敵級）
- 移動速度60
- Lv.MAX強襲型の速度50より高速
- 出現時に戦闘全体を停止
- 専用演出中に場の味方をすべて消去
- その後、黒いBOSS出現演出を再生して戦闘再開
- 味方拠点・Energy・召喚クールタイム・恒久進行は消去しない

## Energy設計

- 上限強化をLv.10からLv.12へ拡張
- 最大Energyは235から265へ増加
- Lv.3で最大130
- 125以上を蓄えてWave 10へ入り、BOSS後に盾・攻撃役を即時再展開する攻略を想定

## 化学表現

O₃の酸化還元属性は高校化学上の強い酸化作用を踏まえています。一方、全味方消去、HP、移動速度、戦闘役割はゲーム独自の表現であり、実際の反応速度・危険性・作用範囲を表しません。
