# Chemion Quest v5.5 通常BGM実装レポート

## 実装内容

- Stage 1〜9共通のChemion Quest専用オリジナルBGM
- 長さ96.0秒、歌詞なし、MP3 96kbps・モノラル
- 効果音とは独立したBGM ON/OFF
- 0〜100％、5％刻みの音量設定
- 設定・ポーズ画面からの切り替え
- localStorage保存とセーブコード移行
- 初回ユーザー操作後の再生開始
- タブ非表示・pagehide時の自動停止と表示復帰時の再開
- PWAオフラインキャッシュ・検査付き公開ZIPへの同梱

## 互換性

- 内部セーブバージョン31を維持
- Stage 1〜9の戦闘データを変更しない
- 基本520問、難問280問、実戦40小問を変更しない
- 効果音設定は従来どおり独立して動作
- BGM設定が存在しない旧端末ではBGM ON・音量35％を既定値とする

## 実装上の判断

UnityやUnreal Engineへ移行せず、既存Web版へ軽量音源を追加しました。通常BGMはデコード済み音声のループ再生を使用し、ゲーム進行・描画ループとは分離しています。Stage 10では通常曲を流用せず、Au・王水専用曲へ切り替える設計を予定します。

## 検査項目

- 音源ファイルの存在・容量
- BGM操作UIと音量UI
- 自動再生制限対応
- 画面非表示時停止
- セーブコードへのBGM設定同梱
- Service Workerキャッシュ
- 公開ZIPのファイル一覧・SHA-256・版整合性
- 既存機能・問題・化学監査・ブラウザ回帰
