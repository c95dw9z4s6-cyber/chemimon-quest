## 開発コマンド

```bash
npm run build
npm run validate
npm run release
```

- `npm run build` — 分割された開発元から公開用`index.html`、`sw.js`、`version.json`、文書を生成
- `npm run validate` — 構文、全問題、問題品質、出題分散、過去機能、PWAを総合検査
- `npm run release` — build・validate・公開ZIP検査を実行し、完全版・移行用ZIP・単一公開ZIPを作成
- `npm run audit` — 過去アップデートの必須機能を監査
- `npm run audit:questions` — 分野・問い方・正答位置・重複を分析
- `npm run audit:chemistry` — 遊離相性、反応式、正式名称、酸化還元BOSS、破損解説を監査
- `npm run test:features` — ポーズ再開、倍速制限、間隔反復、Stage 5実績、セーブ移行を回帰検査
- `npm run package:site` — 公開に必要なファイルだけを`chemion-release.zip`へまとめ、manifestとSHA-256を生成
- `npm run verify:package` — 単一公開ZIPの完全性と版整合性を検査
- `npm run test:package` — 破損・欠落・余分な旧ファイル・危険なパス・版不一致の拒否試験

生成済みの`index.html`を直接編集しないでください。

## GitHubへの公開

### v5.2への初回移行

公開方式を単一ZIPへ切り替えるため、v5.2だけは次の3ファイルを同じコミットで上書き・追加します。

- `.github/workflows/pages.yml`
- `scripts/verify_release_package.py`
- `chemion-release.zip`

`chemion-release.zip`は展開せず、そのままリポジトリ直下へ置きます。以前の`index.html`や`src`などが残っていても、新しいActionsは公開に使いません。

1. 同梱の`chemion_v6_1_github_migration`フォルダーを開く。
2. 上の3ファイルを、フォルダー構成を保ったままGitHubへアップロードする。
3. 3ファイルを同じコミットに含める。
4. Actionsの`Verify package and deploy Chemion Quest`が緑になることを確認する。
5. 公開URLで`Chemion Quest v6.1`を確認する。

途中でファイルが欠けても、検査が失敗して新しい版は公開されず、現在の正常版が維持されます。

### v5.3以降

通常更新では原則、リポジトリ直下の`chemion-release.zip`を新しい同名ファイルへ置き換えるだけです。

Actionsは次を自動確認します。

- ZIPを毎回空の領域へ展開し、前版の残存ファイルを混ぜない
- 宣言されたファイルがすべて存在し、余分なファイルがない
- 全ファイルのSHA-256とバイト数が一致する
- 重複ファイル、危険なパス、破損ZIPを拒否する
- `version.json`、`index.html`、Service Worker、manifestの版番号が一致する
- すべて合格したときだけGitHub Pagesへ一括公開する

`npm run package:site`で単一公開ZIPを生成し、`npm run test:package`で破損・欠落・余分な旧ファイル・版不一致を拒否できるか検査できます。

## 要望一覧の管理方式

v4.45以降では、匿名認証を含むログイン済みの全ユーザーが、すべての要望を「実装済み／検討中」へ変更し、削除できます。`requestAdmins`の登録や管理者IDの設定は不要です。

この権限変更を反映するため、GitHubへの通常更新に加えて、Firebase Consoleの`Firestore Database → ルール`で、同梱の`firestore.rules`を貼り付けて公開してください。削除は元に戻せないため、ゲーム内で確認画面を表示します。

## v6.1以降の確定構想

- Stage 11以降で低確率出現する最初の貴金属系希少敵はPt（白金）です。
- 通常攻撃は1ヒットごとに必ず1ダメージで、多段攻撃は各ヒットが1ダメージとして通ります。多段攻撃が有利なのは意図した特徴です。
- 王水の攻撃だけは1ダメージ制限を無視し、通常のダメージ計算でPtへ通ります。HCl・HNO₃単体は例外にしません。
- 通常の変異体システムはまだ実装せず、Ptは別枠の希少敵として扱います。
